FAQ
Questions about
development.
Straight answers about how we build, what's included, and how it all works.
What technology do you use to build sites? +
We build with clean HTML, CSS, and JavaScript — no page builders, no WordPress bloat. Your site is lean, fast, and easy to host anywhere. For e-commerce we use Shopify. For content-heavy sites we can integrate a lightweight CMS so you can update content yourself.
Can I update the site myself after launch? +
Yes — on the Care plan, just email us with any changes and we'll handle it within 24 hours. If you'd like to edit yourself, we can set up a CMS so you can update text, images, and menu items without touching any code.
How fast will my website load? +
We target a Google PageSpeed score of 90+ on all sites — achieved through clean code, optimised images, minimal dependencies, and fast Vercel hosting. A fast site is critical for both user experience and SEO ranking.
Will my site show up on Google? +
Every site includes technical SEO foundations — proper meta tags, structured data, sitemap, Google Search Console setup, and Google Business profile configuration. This gets you indexed correctly from day one.
What if I want to add more pages or features later? +
No problem. Additional pages can be added at any time — typically $150–$250 per page depending on complexity. Features like booking integrations, contact forms, menu systems, or e-commerce can be quoted and added at any stage.
Who owns the website once it's built? +
You do. Once paid for, you own the code, the design, and all the content. If you cancel the Care plan, we'll help you migrate to your own hosting. No lock-in, no ransom.
What do I need to provide to get started? +
Not much — your logo (if you have one), some photos, and an idea of your brand colours and tone. If you don't have photos, we can advise on free resources. We handle everything else: layout, code, hosting, and launch.
Do you work with businesses outside Brisbane? +
Our focus is Brisbane-based businesses, as we know the local market well. We do occasionally work with Australian businesses outside Brisbane — get in touch and we can have a conversation.
